# Project-3-Online-auction-system

Tutorial on how to make this: https://github.com/gpouilloux/auction-system


Things this project has to it:

Object-Oriented Programming: Classes, inheritance, encapsulation, and object interactions

Threading: Live auction timers and AI bot behavior using Python threads

Polymorphism: Different bot personalities acting on the same Auction interface

Magic Methods: Comparison logic for bids (max() with custom key functions)

CLI Interactivity: Accepting user commands and updating real-time outputs

Project Structuring: Three-file architecture with clear separation of concerns

Testing: Basic unit tests for User and Auction functionality

External Packages: Learning colorama and tabulate to improve user experience
